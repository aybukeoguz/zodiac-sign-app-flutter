## Kurulum

Flutter SDK'nın kurulu olduğundan emin olun.
Depoyu klonlayın: 'git clone [repo bağlantısı]'.
Bağımlılıkları yükleyin: 'flutter pub get'.
Uygulamayı çalıştırın: 'flutter run'.
